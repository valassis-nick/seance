Seance 1.0.0

A Medium-inspired theme for Ghost.

Check out maxjackson.online for a very good example of Seance in action :D.

Features

  • Mobile-first design
  • Fully responsive layout
  • Compatible with Ghost 1.14.1
  • Article header images
  • Automatic figcaption generation from image alt tags
  • Extremely valid HTML5/CSS3
  • WAI-ARIA & Rich Snippets(microdata) roles
  • Static page support
  • Featured article support
  • Google Analytics integration
  • FontAwesome
  • Highlight.js integration for Syntax highlighting
  • OpenGraph & Twitter Cards meta’s
  • Free & open source font usage

Installation:

  1. Upload your Zip file via the Ghost admin panel.
  2. Switch your theme to the seance theme.
  3. Have fun!

Enabling Disqus Comments

Follow the instructions in partials/comment-list.hbs to enable Disqus comments on your posts.
